sinatraxscratch
===============

Prerequisites
-------------
- Ruby 1.9.3

Inport Extension
----------------
On Scratch, select **File** menu with holding shift-key clicked and select **Import Experimental Extension**, then, import **example.json**.

Run the server
---------------
Install the gem:

<code>
gem install sinatra
</code>

And run with:

<code>
ruby app.rb -p 49001
</code>
